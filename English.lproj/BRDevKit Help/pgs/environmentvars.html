Recognised Environment Variables

The test harness application recognises a few key environment variables which enable you to control some of its inner processes. This list may grow with time and new feature requests, but at the present there are two recognised variables:

BDKStartupWithRootItem
Set the value to be the path to a specific appliance plugin, and that plugin will be launched automatically upon startup of the test harness. The path supplied is used as-is, and is therefore considered relative to the working directory. This is designed to be used within the Xcode Custom Executable setup.

BDKSkipIntroMovie
Set the value of this variable to YES (case insensitive) or 1 to make the BackRow window skip the Apple TV intro movie when it opens.
