Using the Xcode Project Template

The Xcode project template provides an easy-to-use starting point for your Apple TV appliance plugin. It provides a basic BRAppliance subclass, fully implemented, and a BRMediaMenuController subclass to use as the basis of your plugin's unique UI.

The project is setup by default to build Intel-only binaries, and is already linked against all necessary frameworks. Only a little application-specific code is needed to fill out the menu.

Open Xcode, and select File->New Project.

In the New Project window, open the Apple TV group and select Appliance.
